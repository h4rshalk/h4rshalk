## 🔍 AuctionIQ — Deep Dive

> *"CV Varun at ₹1.7Cr delivers 47x more value per crore than players priced at ₹17Cr. The data doesn't lie."*

**What I built:**
- 📊 Processed **287,513 ball-by-ball records** across **19 IPL seasons** (2007–2026)
- 🏏 Designed **Batting Performance Index (BPI)** and **Bowling Performance Index (BoPI)** — weighted composite metrics
- 💰 Built **Auction Valuation Model** — identifies undervalued, overvalued, premium, and budget players
- 📈 **3-page Power BI dashboard** with valuation matrix, player rankings, and IPL evolution trends

**Key findings:**
- 🥇 **H Klaasen** — #1 Batter (BPI: 86.12)
- 🎯 **JJ Bumrah** — #1 Bowler (BoPI: 83.65)  
- ⚡ **SP Narine** — Only TRUE Allrounder in top 15
- 💎 **CV Varun** — Most undervalued (₹1.7Cr, value ratio: 47.28x)
- ⚠️ **RG Sharma** — Most overvalued (value ratio: 4.07x)
